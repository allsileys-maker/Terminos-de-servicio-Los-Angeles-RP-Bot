BOT INFO

Términos de Uso del Bot

1. Aceptación de los Términos

Al utilizar este bot dentro de cualquier servidor de Discord, el usuario acepta cumplir con los presentes Términos de Uso.

2. Uso del Bot

El bot está diseñado para proporcionar funciones de utilidad, administración, entretenimiento, economía virtual, automatización o roleplay dentro de servidores de Discord.

3. Cumplimiento con Discord

El uso del bot debe respetar en todo momento los términos y normas de Discord.

4. Uso Adecuado

  • No spam de comandos
  • No abusar del sistema
  • No realizar ataques o pruebas de estrés
  • No automatizar carga excesiva
  • No usar el bot para acoso

5. Abuso de Comandos

El uso excesivo de comandos puede resultar en restricciones automáticas.

6. Moderación

Los administradores del servidor pueden restringir el uso del bot.

7. Sistema de Datos

  • ID de usuario
  • ID de servidor
  • Configuraciones
  • Datos de economía
  • Historial de comandos

8. Privacidad

No se recopilan datos personales fuera de la información necesaria de Discord.

9. Disponibilidad

No se garantiza disponibilidad continua del bot.

10. Cambios en el Bot

El desarrollador puede modificar el bot en cualquier momento.

11. Responsabilidad

  • Mal uso del bot
  • Pérdida de economía virtual
  • Errores de configuración
  • Decisiones de administradores

12. Seguridad

Intentar explotar vulnerabilidades constituye una violación grave.

13. Restricción de Acceso

El bot puede bloquear usuarios o servidores abusivos.

14. Eliminación de Datos

Administradores pueden solicitar eliminación de datos.

15. Suspensión

El desarrollador puede suspender el bot temporal o permanentemente.

16. Uso en Múltiples Servidores

Estos términos aplican a todos los servidores donde esté el bot.

17. Modificaciones

Los términos pueden cambiar en cualquier momento.

18. Contacto

Contactar al desarrollador para errores o abuso.

19. Terminación

El acceso puede ser revocado si se violan los términos.

20. Aceptación Final

Al usar el bot se aceptan estos términos.

Aclaraciones

Este bot es una herramienta diseñada para ayudar en la administración y funcionamiento de servidores de Discord.

Las funciones del bot pueden cambiar, mejorar o eliminarse dependiendo del desarrollo del proyecto.

El uso del bot siempre está sujeto a las reglas del servidor donde se utilice.

Servidores

El bot puede ser utilizado en múltiples servidores simultáneamente.

Cada servidor mantiene sus propias configuraciones y administración.

Los administradores de cada servidor son responsables del uso que los usuarios hagan del bot.

Cosas a saber para usar el bot

  • No spamear comandos
  • Leer la ayuda del bot
  • Respetar reglas del servidor
  • Reportar errores al desarrollador
  • No intentar explotar bugs
